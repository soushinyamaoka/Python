COOP連携API（coop-api）プロジェクトの直近1週間の作業内容を会社報告用にまとめてください。

## 手順

1. `git log --oneline --since="7 days ago" --no-merges` で直近1週間のコミットを取得する
2. 各コミットの変更内容を `git show --stat <hash>` で確認する
3. 必要に応じて `git diff` で具体的な変更内容を確認する

## 出力フォーマット

以下の形式で出力してください:

```
【COOP連携API】<期間>
<作業内容を100字程度で要約>
```

## ルール
- 100字程度（80〜120字）に収める
- 技術用語は最小限にし、機能面での説明を優先する
- 「〜を実装」「〜を修正」「〜を追加」のような動詞で締める
- コミットがない場合は「該当期間のコミットはありません」と報告する
- 箇条書きではなく、1文〜2文の文章でまとめる

## 出力例
```
【COOP連携API】3/18〜3/24
注文食材から1週間分の献立を自動作成するAPIを新規実装し、食材の消費管理（残量追跡）機能と調理キット・冷凍食品の自動判定ロジックを追加した。
```
